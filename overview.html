This is a Java library built on top of Java Simple Argument Parser (JSAP) that allows the use of a default "configuration file" using the --config-file option.

This package makes it easy for one to implement a configuration file system that combines the flexibility of the command line and persistence of an actual configuration file.

See the main class {@link edu.cmu.cs.ark.yc.config.AppConfig} and {@link com.martiansoftware.jsap.JSAP} for more information on using it. There is also a {@link edu.cmu.cs.ark.yc.config.DemoApp} included.
